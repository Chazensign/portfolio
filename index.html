ActvlySocial is a social media platform for people with active lives. It was built using React.js, Express, Massive, Node.js, PostgreSQL, Redux, Sockets.io, Google Geocoding API, Google Distance Matrix API, Google Maps API and the RedLine Zipcode API.

Friday Night Lite is a site where fans or parents can keep up with high school football games when they are unable to be there. Friday Night Lite was built using React.js, Express, Mongoose, MongoDB and Sockets.io for live updates.

Dixie Lacrosse was built for a client that wanted to have a web application that would allow them to update all the site information. Whether they aquire new sponsors, want to update the team picture or forms, add and remove events from the calendar or edit the contact information.

The site clone is to show my ability with CSS and Media Queries based off of a provided template, the original site that was cloned can be found here.
